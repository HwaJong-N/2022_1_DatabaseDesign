## 개발환경

* Oracle DB

* iSQL* Plus

## 프로젝트 소개

2022학년도 1학기 데이터베이스설계 과목을 수강하면서 1학기 동안 했던 팀프로젝트이며 수행 내용은 아래와 같다

1. 주제 및 기능 선정

2. 2. 요구사항 수집

3. 업무 요구분석 명세서 및 트랜잭션 명세서 작성

4. 개체, 속성, 관계 식별 및 ERD 작성

5. 각 업무별 트랜잭션 모델링

6. 제약조건 정의

7. 개체 및 관계 릴레이션 스키마 작성 및 단순화

8. 무결성 제약조건 정의

9. 모든 릴레이션이 3NF 를 만족하도록 정규화 

10. 개체, 관계에 대한 내부 스키마 작성

11. 테이블 생성 및 데이터 삽입

12. 트리거 작성
